# Projeto: Lista de Tarefas (To-Do List)

## Descrição
Desenvolvimento de uma aplicação interativa para gerenciamento de tarefas diárias. O projeto foca na manipulação dinâmica de elementos da interface e na gestão de estados de itens (pendente/concluído).

## Tecnologias Utilizadas
- HTML5: Estrutura da interface de entrada e exibição.
- CSS3: Estilização de estados visuais (tarefas ativas e finalizadas) e layout responsivo.
- JavaScript (ES6+): Lógica para adição, remoção e alternância de status das tarefas via manipulação do DOM.

## Funcionalidades
- Adicionar tarefa: Permite inserir novas atividades na listagem.
- Marcar como concluída: Altera visualmente a tarefa para indicar finalização.
- Excluir tarefa: Remove itens específicos da lista.
- Validação: Impede a adição de tarefas vazias.
